# 🌱 EcoShare – Smart Food Waste Management & Redistribution System

EcoShare is a full-stack web application designed to reduce food wastage by connecting students, hostels, restaurants, cafeterias, NGOs, and administrators through an intelligent food management and redistribution platform.

The system predicts meal demand, manages food inventory, tracks leftover food, and ensures surplus food is redistributed to nearby NGOs before it expires using Data Structures and Algorithms (DSA).

# ⚡ DSA Implementation

The project demonstrates real-world implementation of Data Structures & Algorithms.

| Data Structure / Algorithm | Purpose |
|----------------------------|---------|
| Hash Map | Student, Inventory & Food Records |
| Priority Queue | Food Expiry Priority |
| Graph | Kitchen → NGO Route Mapping |
| Dijkstra Algorithm | Shortest Route Calculation |
| Greedy Algorithm | Best NGO Selection |
| Queue | Food Pickup Scheduling |
| Stack | Undo Recent Operations |
| Linked List | Booking History |
| Binary Search | Fast Student Search |
| Trie | Smart Search Suggestions |
| Heap | Highest Priority Food Donations |
| Sorting Algorithms | Food & Donation Ranking |
